vertex color

Only UniGLTF / UniUnlit supports vertex colors.

  • Mesh contains vertex colors
  • Material is a Unlit

In some cases, importing will apply the vertex color. If you apply Unity’s unlit system material to a model that does not require vertex color and export it, The next time you import, the colors may change unintentionally. In this case, you can export Mesh without vertex colors by enabling RemoveVertexColor at the time of export.
