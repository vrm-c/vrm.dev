# Runtime Import

glb vrm vrma などのファイルを入力、
UnityEngine.GameObject のヒエラルキーを出力とする手順です。
2ステップに別れています。

はじめにファイルか中間データ[UniGLTF.GltfData](/api/runtime-import/gltfdata)  の生成をします。この手順は glb, vrm-0.x, vrm-1.0, vrm-animation で共通です。

次に glb, vrm-0.x, vrm-1.0, vrm-animation などの glTF 拡張に応じた
GameObject 構築手順を実行します。
