# VRM-1.0

## VRM 1.0 とは

2018 年に発表した 3D アバターファイルフォーマット「VRM」において、
利用が進むにつれて判明した課題やエラーを修正し、 今後必要になると考えられる機能を追加した規格となります。

従来の VRM は今後 VRM 0.x と分類され非推奨となり、今後は VRM 1.0 が正式な規格となる予定です。

VRM 1.0 は、2022 年 9 月にリリースされました。
